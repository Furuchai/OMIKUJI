# 運試しおみくじ (Omikuji App)

ボタンを押すとランダムで運勢が表示される、シンプルなおみくじアプリです。
JavaScriptの配列（Array）と乱数生成の仕組みを学習するために作成しました。

## 🎮 デモ (Webアプリ)
以下のリンクから、実際に運試しができます。今日の運勢を占ってみてください！

[👉 **運試しおみくじを引く**](https://furuchai.github.io/OMIKUJI/)

## 📸 スクリーンショット
![おみくじの結果画面](OMIKUJI.png)

## 🕹️ 遊び方
1. **「おみくじを引く」** ボタンをクリックします。
2. ランダムに結果（大吉、吉、凶など）が表示されます。
3. 結果によって文字の色が変わる演出もあります。
4. 何度でも引き直すことができます。

## 🛠️ 使用技術
* HTML5
* CSS3
* JavaScript (配列操作, Math.random)
